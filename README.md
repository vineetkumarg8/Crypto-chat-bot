### Prerequisites
- Node.js (16.0+)
- npm or yarn

### Development

# Install dependencies
npm install

# Start development server
npm run dev

Open `http://localhost:5173` in your browser.

### Build
npm run build


## Usage Examples

- "What's Bitcoin trading at?"
- "Show chart for Ethereum"
- "I have 2 BTC"
- "What's my portfolio worth?"
- "Show me trending coins"
